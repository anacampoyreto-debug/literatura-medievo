\# 📜 Literatura Medieval Interactiva



Proyecto web interactivo sobre \*\*Literatura Medieval\*\* pensado para trabajar contenidos de \*\*1º ESO\*\* de forma visual, práctica y autocorregible.



La página incluye cuatro bloques temáticos con preguntas tipo test, retroalimentación inmediata, modo práctica y modo examen.



\---



\## ✨ Contenido del proyecto



\### Bloques incluidos

\- \*\*Bloque 1:\*\* Poesía popular (lírica y épica)

\- \*\*Bloque 2:\*\* Poesía culta (clerecía y Manrique)

\- \*\*Bloque 3:\*\* Prosa y teatro (Don Juan Manuel y \*La Celestina\*)

\- \*\*Bloque 4:\*\* Laboratorio técnico (métrica y figuras)



\### Funciones de la web

\- Navegación por bloques

\- Preguntas tipo test autocorregibles

\- Retroalimentación en cada respuesta

\- Modo práctica

\- Modo examen

\- Barra de progreso

\- Contador de preguntas

\- Contador de aciertos

\- Resultado final por bloque

\- Diseño visual medieval adaptado a móvil y ordenador



\---



\## 📁 Archivos del proyecto



\- `index.html` → estructura principal de la web

\- `style.css` → diseño visual y estilo

\- `data.js` → base de datos con todos los bloques, preguntas y respuestas

\- `README.md` → información del proyecto



\---



\## ▶️ Cómo usarlo



\### En local

1\. Descarga todos los archivos.

2\. Guárdalos en la misma carpeta.

3\. Abre `index.html` en tu navegador.



\### En GitHub Pages

1\. Crea un repositorio en GitHub.

2\. Sube estos archivos:

&#x20;  - `index.html`

&#x20;  - `style.css`

&#x20;  - `data.js`

&#x20;  - `README.md`

3\. Ve a \*\*Settings > Pages\*\*.

4\. En \*\*Source\*\*, selecciona la rama principal y la carpeta raíz (`/root`).

5\. Guarda los cambios.

6\. GitHub generará una URL pública para abrir la web.
